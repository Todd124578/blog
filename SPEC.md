# 个人博客规范文档

## 1. 项目概述

- **项目名称**: 个人博客
- **项目类型**: 静态博客网站
- **核心功能**: 展示个人文章、简介、相册的简约博客
- **目标用户**: 访客、读者

## 2. UI/UX 规范

### 布局结构

- **导航栏**: 固定顶部，包含Logo和导航链接
- **首页**: Hero区域 + 文章列表 + 关于简介
- **文章页**: 标题 + 日期 + 正文内容
- **页脚**: 版权信息和社交链接

### 响应式断点

- 移动端: < 768px (单列布局)
- 平板: 768px - 1024px
- 桌面: > 1024px (最大宽度1200px居中)

### 视觉设计

**配色方案** (暗色主题)
- 背景色: #0f0f0f (深黑)
- 卡片背景: #1a1a1a
- 主色调: #e8c547 (金色)
- 次色调: #f5f5f5 (浅白)
- 文字主色: #e0e0e0
- 文字次色: #888888

**字体**
- 标题: "Playfair Display", serif
- 正文: "Source Sans Pro", sans-serif
- 代码: "Fira Code", monospace

**间距系统**
- 基础间距: 8px
- 小间距: 16px
- 中间距: 24px
- 大间距: 48px

**视觉效果**
- 卡片阴影: 0 4px 20px rgba(0,0,0,0.3)
- 悬停效果: transform: translateY(-4px)
- 过渡动画: 0.3s ease

### 组件

1. **导航栏**
   - Logo文字
   - 导航链接(首页/文章/关于)
   - 移动端汉堡菜单

2. **Hero区域**
   - 大标题
   - 副标题/个人简介
   - 社交链接图标

3. **文章卡片**
   - 封面图(可选)
   - 标题
   - 日期
   - 摘要
   - 标签

4. **文章内容**
   - 标题
   - 元信息(日期/分类)
   - Markdown渲染的正文
   - 代码高亮

5. **关于页面**
   - 头像
   - 个人简介
   - 技能标签

6. **页脚**
   - 版权文字
   - 社交图标

## 3. 功能规范

### 核心功能

1. **首页展示**
   - 显示最新文章列表
   - 点击跳转到文章详情

2. **文章系统**
   - Markdown格式文章
   - 代码高亮显示
   - 分类标签

3. **关于页面**
   - 个人简介
   - 技能展示

4. **响应式设计**
   - 完美适配手机/平板/桌面

### 文章数据(示例)

- 3篇示例文章
- 包含不同分类和标签

## 4. 技术实现

- 纯HTML + CSS + JavaScript
- 无需后端
- 使用本地Storage模拟数据
- CSS变量管理主题
